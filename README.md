Inventory Management System
A comprehensive warehouse management solution built with ASP.NET Core MVC.

Features

-Multi-Warehouse Management - Track inventory across multiple locations
-Product & Supplier Management - Complete catalog and supplier relations
-Order Processing - Supply orders, release orders with stock validation
-Inter-Warehouse Transfers - Move products between warehouses
-Analytics & Reports - Warehouse, product, movement, aged inventory, and expiry reports
-Real-time Stock Monitoring - Live inventory tracking with automated validation

Tech Stack

-ASP.NET Core MVC
-Entity Framework Core
-SQL Server
-Bootstrap UI



Installation

-bashgit clone https://github.com/abdelrahmanelsaadany1/Inventory_Management_System.git
-cd Inventory_Management_System
-dotnet restore
-dotnet ef database update
-dotnet run

Usage

-Set up warehouses and products
-Add suppliers and customers
-Process supply/release orders
-Transfer inventory between warehouses
-Generate comprehensive reports
